# Install
This project requires [NodeJs](http://nodejs.org/), [Ruby](http://www.ruby-lang.org), [Compass](http://compass-style.org/), [Webpack]


```bash
npm install webpack -g
npm install

```

# Run Project

npm start


# Build Project

npm run build
